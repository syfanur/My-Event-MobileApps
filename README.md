# My-Event-MobileApps
MyEvent adalah aplikasi berbasis mobile yang bertujuan untuk melakukan pembelian tiket sebuah acara. Aplikasi ini menggunakan bahasa pemrograman Java
dan mmenggunakan Firebase sebagai databasenya.
